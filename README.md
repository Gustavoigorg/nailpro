# 💅 NailPRO

**PWA Offline** para manicures e salões gerenciarem agenda e clientes.

## ✨ Funcionalidades
- 📅 Agenda drag & drop
- 👩‍🦰 CRM clientes + histórico preferências
- 💰 Receita por serviço + relatórios
- 📱 **Instalável** Android/iOS
- 🌐 **Offline total**

1. Acesse https://nailprolucro.netlify.app
2. Menu > "Adicionar à tela inicial"
